Robot Operating System (ROS) Workshop

ROS (Robot Operating System) is like a toolbox for building and controlling robots. It helps researchers and companies make robots do all sorts of tasks, from self-driving cars to farming and healthcare robots. ROS provides a way to talk to the robot's hardware, send messages between different parts of the robot's brain, manage pieces of code neatly, and use useful tools. It's popular because it's flexible and used in many fields.

Arduino/Python Robot

An Arduino/Python Robot Workshop is a hands-on educational event where participants construct a robot using Arduino, a beginner-friendly hardware platform, and program it with Python, a user-friendly programming language. The workshop guides attendees through the process of assembling the robot, connecting sensors and motors, and then writing Python code to control the robot's movements and interactions with its surroundings. This workshop offers an approachable way for beginners to learn about both the hardware and software aspects of robotics and gain practical experience in building and programming robots.
